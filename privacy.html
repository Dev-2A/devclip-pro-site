Privacy Policy

DevClip Pro — Chrome Extension · Last updated: April 2026

1. Introduction

DevClip Pro ("we", "our", "the extension") is a Chrome browser extension that helps developers save, organize, and search code snippets from web pages. This privacy policy explains what data we collect, how we use it, and your rights regarding your data.

2. Data We Collect

2.1 Free Plan Local Only

When using the Free plan, all data is stored locally on your device using the Chrome Storage API. We do not collect, transmit, or store any of your data on our servers. This includes:

  • Code snippets you save
  • Tags you create
  • Your extension settings (theme, preferences)

We have zero access to your Free plan data.

2.2 Pro Plan Cloud Sync

When you upgrade to Pro and opt into cloud sync, the following data is stored on our servers (Supabase, hosted on AWS):

  • Email address — for account creation and login
  • Code snippets — title, code content, detected language, source URL
  • Tags — tag names and associations
  • Subscription status — managed by Lemon Squeezy

3. Data We Do NOT Collect

  • Browsing history or web activity
  • Personal information beyond email
  • Financial information (handled entirely by Lemon Squeezy)
  • Location data
  • Device identifiers or fingerprints
  • Analytics or usage tracking data

4. How We Use Your Data

  • Authentication — to identify you and provide access to your account
  • Cloud sync — to synchronize your snippets across devices
  • AI tagging — when you request AI tag suggestions, your code snippet is sent to the Anthropic Claude API for analysis. We do not store the AI analysis results on our servers beyond the suggested tags.

5. AI Processing

When you use the AI auto-tagging feature (Pro plan only):

  • Your code snippet is sent to Anthropic's Claude API for analysis
  • Only the code content, detected language, and snippet title are sent
  • We do not use your code to train AI models
  • Anthropic's data retention policies apply to the API request

6. Third-Party Services

  • Supabase (database & authentication) — Privacy Policy
  • Lemon Squeezy (payment processing) — Privacy Policy
  • Anthropic (AI processing) — Privacy Policy

7. Data Storage & Security

  • Cloud data is stored on Supabase (AWS infrastructure, Seoul region)
  • All data transmission uses HTTPS encryption
  • Database access is protected by Row Level Security (RLS) — each user can only access their own data
  • API keys and secrets are stored as environment variables, never in client code

8. Data Retention & Deletion

  • Free plan: Data exists only on your device. Uninstalling the extension removes all data.
  • Pro plan: You can delete your account and all associated data by contacting us. Upon subscription cancellation, your data remains accessible until you request deletion.

9. Your Rights

You have the right to:

  • Access your data (available through the extension's export feature)
  • Delete your data (contact us or use the "Delete all data" option in settings)
  • Export your data (JSON or Markdown export in the extension)
  • Cancel your subscription at any time

10. Children's Privacy

DevClip Pro is not intended for children under 13. We do not knowingly collect data from children under 13.

11. Changes to This Policy

We may update this privacy policy from time to time. We will notify users of significant changes through the extension or via email.
